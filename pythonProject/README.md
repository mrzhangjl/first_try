# YOLO5 物体识别可视化工具

这是一个使用YOLO5实现的物体识别GUI程序，提供了友好的界面来展示YOLO在运行过程中的数据可视化功能，支持大多数YOLO参数调节，并包含YOLO优化和学习方法的指南。

## 功能特点

### 1. 物体检测功能
- 支持图片、视频和摄像头三种输入源
- 实时显示检测结果，包括边界框和置信度
- 可保存检测结果图片和数据

### 2. 参数调节功能
- 置信度阈值调节：控制检测结果的可信度
- IOU阈值调节：控制非最大抑制的严格程度
- 最大检测数量设置：限制每张图片/帧的最大检测物体数
- 图像大小调整：影响检测精度和速度
- 类别过滤：可选择只检测特定类别的物体

### 3. 数据可视化功能
- 类别分布饼图：直观展示各类物体的检测数量比例
- 检测频率折线图：显示随时间变化的检测数量趋势
- 置信度分布图：展示检测结果的置信度分布情况
- 检测速度折线图：实时监控FPS变化

### 4. 优化和学习方法
- 性能优化指南：包括模型选择、输入尺寸优化、批量推理等
- 训练技巧：数据准备、数据增强、超参数调整等
- 部署指南：模型导出、不同平台部署、实时推理优化等

## 安装说明

### 1. 环境要求
- Python 3.8+
- 推荐使用CUDA兼容的GPU以加速推理

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 下载YOLO5模型
默认使用`yolov5s.pt`模型，您可以从[Ultralytics YOLOv5官方仓库](https://github.com/ultralytics/yolov5/releases)下载预训练模型。

## 使用方法

### 1. 基本流程
1. 运行主程序：`python example.py`
2. 在"物体检测"标签页中，点击"使用默认模型"或通过"浏览"选择自定义模型文件
3. 选择输入源（图片、视频或摄像头）
4. 对于图片/视频，点击"选择文件"按钮选择要处理的文件
5. 点击"开始检测"按钮进行检测
6. 检测结果将实时显示在右侧窗口

### 2. 参数调节
- 在"参数调节"标签页中，可以调整各种检测参数
- 调整后参数会立即生效（部分参数需要点击"应用"按钮）

### 3. 数据可视化
- 在"数据可视化"标签页中，选择要查看的可视化类型
- 检测过程中或检测完成后，点击"刷新"按钮更新可视化图表

### 4. 保存结果
- 检测完成后，点击"保存结果"按钮
- 结果将保存为图片文件（包含检测框）和CSV文件（包含详细检测数据）
- 保存在`detection_results`目录下

## 注意事项

1. 首次加载模型可能需要一些时间，特别是通过torch.hub下载时
2. 视频和摄像头检测时，可随时点击"停止检测"按钮中断
3. 在GPU环境下运行可获得更好的性能
4. 对于大型视频文件，处理可能需要较长时间
5. 如果未找到默认模型文件，请先下载`yolov5s.pt`或使用浏览按钮选择模型文件

## 优化建议

1. 根据需要选择合适的模型大小：
   - `yolov5s.pt`：速度最快，适合实时应用
   - `yolov5m.pt`：平衡速度和精度
   - `yolov5l.pt`、`yolov5x.pt`：精度更高，但速度较慢

2. 调整图像大小以平衡速度和精度：
   - 降低图像分辨率（如320x320）可提高速度
   - 提高图像分辨率（如832x832）可提高精度

3. 对于特定场景的检测，可考虑使用自定义训练的模型以获得更好的效果
